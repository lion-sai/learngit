Git is a version control system.
Git is free software.
create a repository
    $ mkdir learngit 
    $ cd learngit  #创建一个名为learngit的目录
    $ pwd   #显示当前目录
    
    $ git init  #目录learngit变成一个Git仓库

添加文件至仓库  e.g. 仓库里有文件 "readme.txt"
    $ git add "readme.txt" #添加
    $ git commit -m "xxx"  #提交，-m后带注释
